# Ultrasonensensor

De ultrasonensensor werkt door de de tijd die nodig was voor het geluid om terug te keren maal de snelheid van het geluid te doen of met code: \
`time * 34 / 1000000` \

Deze werd gebruikt om het blik te vinden.
